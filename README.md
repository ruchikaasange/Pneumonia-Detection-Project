# DS-N-WE-H-B38 Final Project: Pneumonia Detection Challenge

An advanced Deep Learning and Data Science project using **Convolutional Neural Networks (CNN)** to detect Pneumonia from Chest X-ray images.

## 🚀 Key Features
- **Deep Learning Model:** CNN architecture built using **TensorFlow/Keras**.
- **Data Augmentation:** Image Preprocessing with Rescaling, Zooming, and Shearing.
- **Interactive UI:** Built an easy-to-use web app using **Streamlit** for real-time predictions.
- **Evaluation Metrics:** Includes training/validation loss graphs, accuracy curves, and confusion matrix.

## 🛠️ Tech Stack
- **Language:** Python
- **Deep Learning:** TensorFlow, Keras
- **Web App:** Streamlit
- **Data Handling:** NumPy, Pillow
- **Visualization:** Matplotlib, Seaborn
